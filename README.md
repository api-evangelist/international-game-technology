# International Game Technology (international-game-technology)

International Game Technology (IGT) is a global gaming company that delivers entertaining and responsible gaming experiences across all gaming channels and regulated segments, including lottery, gaming machines, sports betting, and digital gaming. IGT does not currently offer a public developer API program; its technical resources are delivered through licensed B2B partner channels.
